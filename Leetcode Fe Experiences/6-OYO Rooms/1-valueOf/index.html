If a == 1 && a == 2 && a == 3, what will be the value of a?

In JavaScript, valueOf is a method that is automatically called by the language when an object is used in a context where a primitive value (like a number, string, or boolean) is expected. The valueOf method allows you to specify how an object should be converted to a primitive value.
